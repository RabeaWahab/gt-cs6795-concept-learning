Three theories of a concept, judged by how they struggle

Prototype, exemplar, and rule models learn the same six category puzzles. The winner isn’t the most accurate one — it’s the one that finds hard what people find hard.

The stimuli

Eight objects, three on/off features, six ways to sort them

Each object is a three-bit code (e.g. 1·0·1). Every structure splits the eight into two categories of four. Colour marks category A vs B; the code beneath is the stimulus.

What the models show

Three results

01

Only the exemplar model learns the human way

With selective attention on, ALCOVE reproduces the human order I < II < {III,IV,V} < VI — including the “unexpectedly easy” Type II.

02

Attention is what makes Type II easy

Freeze attention and Type II collapses from second-easiest to nearly the hardest. Learning which features matter is the mechanism — flip the switch above to see it.

03

The others fail, informatively

The prototype model can’t represent XOR or parity — it never learns Types II or VI. The rule model nails Type I in one error but finds Type II hard: the opposite of people.
